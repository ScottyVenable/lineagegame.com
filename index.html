Guide your ancestors. Shape their destiny. Forge a legacy.

Lineage: Ancestral Legacies is an evolutionary RTS and colony simulation where you nurture a tribe of early hominids, guiding them from survival to civilization through genetics, culture, and emergent storytelling.

Game Overview

  • Genre: Evolutionary RTS / Colony Simulation Strategy
  • Perspective: 2D Top-Down Pixel Art
  • Platform: PC (Windows, Mac/Linux planned)
  • Engine: GameMaker Studio 2
  • Audience: Strategy & simulation fans, emergent storytelling lovers, and legacy-builders

Core Pillars

Legacy & Progression

// Successive generations inherit traits, skills, and knowledge. This teaches about inheritance and progression systems in games.

Emergent Storytelling

// Pops (villagers) have needs, skills, and AI-driven behaviors, creating unique stories each playthrough.

Procedural World

// Each world is generated uniquely, encouraging replayability and teaching procedural generation concepts.

Language Evolution

// In-game language system evolves as the lineage advances, demonstrating procedural content and player agency.

Key Features

  • Procedural World Generation: // Generates a unique world each playthrough for replayability.
  • Dynamic Pop System: // Pops (villagers) have needs, skills, and AI-driven behaviors.
  • Resource Management: // Players gather, store, and use resources to grow their lineage.
  • Legacy Progression: // Successive generations inherit traits and skills, teaching about inheritance in games.
  • Language Evolution: // In-game language system evolves as the lineage advances.
  • Event System: // Random and scripted events challenge the player and add variety.

Gameplay Loop

  • Core Loop: // Gather, build, survive, and progress your lineage.
  • AI Pops: // Pops act autonomously, teaching about AI in games.
  • Building System: // Players place and upgrade structures for their tribe.
  • Seasons & Events: // Dynamic world events affect strategy and planning.
  • Exploration: // Discover new biomes and resources, encouraging curiosity.

Progression

  • Generational Legacy: // Each generation inherits traits, skills, and knowledge.
  • Skill Trees: // Unlock new abilities and technologies over time.
  • Milestones: // Achieve goals to advance your lineage and unlock rewards.
  • Difficulty Scaling: // The game becomes more challenging as you progress, teaching about balancing.

Language System

Proto-Language System

  • Word Creation: // Pops invent words for new concepts, showing procedural content generation.
  • Language Spread: // Words spread through the population, simulating real-world language evolution.
  • Player Influence: // Players can influence language by naming things, teaching about player agency.

Roadmap
